# 최병수 개인 포트폴리오 웹사이트

Bies Worldwide 대표 최병수의 공식 개인 웹사이트

## 🌟 소개
18년간의 국제무역 경험을 가진 글로벌 비즈니스 전문가의 포트폴리오 사이트

## 🛠️ 기술 스택
- React + TypeScript
- Node.js + Express
- Tailwind CSS
- Drizzle ORM

## ✨ 주요 기능
- 한국어/영어 지원
- 반응형 디자인
- 연락처 폼
- 경력 타임라인
